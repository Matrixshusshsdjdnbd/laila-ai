# Privacy Policy Hosting — Quick Guide

## How to host your Privacy Policy for FREE using GitHub Pages:

### Step 1: Create a GitHub Repository
1. Go to https://github.com/new
2. Repository name: `laila-ai-privacy`
3. Set to **Public**
4. Click "Create repository"

### Step 2: Upload the Privacy Policy
1. Click "Add file" → "Upload files"
2. Upload the file `privacy-policy.html` from the `frontend/` folder
3. Rename it to `index.html` when uploading (or create a new file called `index.html` and paste the content)
4. Commit the file

### Step 3: Enable GitHub Pages
1. Go to repository Settings → Pages
2. Under "Source", select **Deploy from a branch**
3. Branch: **main**, folder: **/ (root)**
4. Click "Save"

### Step 4: Get Your URL
After 1-2 minutes, your privacy policy will be live at:
```
https://YOUR-GITHUB-USERNAME.github.io/laila-ai-privacy/
```

### Step 5: Use in Google Play
Paste this URL in Google Play Console → Store Listing → Privacy Policy URL

---

That's it! Free, permanent, professional privacy policy page.
